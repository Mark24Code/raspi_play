raspi_play

==========

####使用方法：
1. 打开终端，cd指令到视频文件所在目录
+ 把play.py，放在这个目录下
+ python play.py 视频文件.mp4 
+ 注释：脚本会搜索该目录下的所有文件，找到视频匹配的字幕，并且播放视频 
  
  
  
####运行环境：
1. 安装python2.7。
2. 系统：Raspberry(树莓派)Raspbian系统。
3. 安装：omxplayer（Raspbian系统自带）
  
####视频格式：
mkv,avi,flv,mp4

####字幕格式：
ssa,ass,smi,srt,sub,lrc,sst,txt,xss,psb,ssb
