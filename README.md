raspi_play
==========

使用方法：
 1：打开终端，cd指令到视频文件所在目录
 2：把play.py，放在这个目录下
 3：python play.py 视频文件.mp4 #注释：脚本会搜索该目录下的所有文件，找到视频匹配的字幕，并且播放视频
  
  
  
说明：
  辅助 omxplayer 播放器，自动匹配字幕。
  环境：Raspberry(树莓派)Raspbian系统。
  基于：omxplayer
  
视频格式：
mkv,avi,flv,mp4

字幕格式：
ssa,ass,smi,srt,sub,lrc,sst,txt,xss,psb,ssb
